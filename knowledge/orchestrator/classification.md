# Ticket Classification Patterns

## Webflow Indicators
- Mentions: blog, CMS, collection, page content, hero, section, FAQ, team member
- References: webflow, website, site, landing page, pricing page
- Actions: update text, add blog post, change image, modify content

## GHL Indicators
- Mentions: workflow, automation, trigger, funnel, pipeline, email sequence
- References: GoHighLevel, GHL, CRM, contacts, deals
- Actions: update workflow, change trigger, modify automation

## Both Platforms
- When a ticket mentions both website content AND workflow changes
- Route sequentially: Webflow first, then GHL

## Unclear
- Generic requests without platform-specific keywords
- Ask for clarification via "Needs Info" status
